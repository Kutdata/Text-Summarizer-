🚀 Text Summarizer with Streamlit  

This is a text summarization application built using Python and Streamlit.  
The app allows users to enter text or a URL and get a summarized version of the content using Sumy’s LSA Summarizer.  

✨ Features  
✅ Summarizes text entered manually or extracted from a given URL  
✅ Supports multiple languages (**Turkish, English, German**) 🌍  
✅ Adjustable summary length (number of sentences) ✂️  
✅ Option to download the summarized text 📥  

 🛠️ Technologies Used  
🔹 Streamlit – Web-based UI  
🔹 Requests – Fetching text from URLs  
🔹 BeautifulSoup – Extracting text from web pages  
🔹 Sumy – LSA-based text summarization  

📝 How It Works
1️⃣ Enter a text or provide a URL 🌐
2️⃣ Choose the language 🏳️
3️⃣ Select the summary length ✂️
4️⃣ Click the Summarize button 🏁
5️⃣ View the generated summary and download it if needed 💾
